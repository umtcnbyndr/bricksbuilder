 Bricks Builder Template Library

Bricks Builder icin hazir template ve section koleksiyonu.

## Klasor Yapisi

- **sections/** - Kategorize edilmis section'lar
  - headers/ - Header template'leri
  - footers/ - Footer template'leri
  - hero/ - Hero section'lar
  - contact/ - Iletisim formlari (44 adet!)
  - features/ - Ozellik grids
  - team/ - Takim section'lari
  - blog/ - Blog section'lari

- **full-sites/** - Tam site template'leri
  - smile/ - Klinik/Dental (9 template)
  - promover/ - Marketing ajans (10 template)
  - uplift/ - Kurumsal site (8 template)
  - cosmetics/ - E-ticaret (10 template)
  - simple/ - SaaS/Startup (8 template)

## Nasil Kullanilir

1. JSON dosyasini ac
2. Tum icerigi kopyala (Ctrl+A, Ctrl+C)
3. Bricks Editor'de sag tik > **Paste**
